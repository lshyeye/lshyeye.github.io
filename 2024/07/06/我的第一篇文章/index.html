使用 hexo+github 搭建个人 blog

文档 | Hexo (将 markdown 转换为 博客形式)

下载 Node.js 注意==版本的匹配==

新建仓库

1.新建一个 Github 仓库，且仓库名字必须为**用户名.github.io**，如果不使用会导致页面出现问题

2.安装 Git

3.获取 SSH key

4.安装 Node.js 与 Hexo

建立一个 github 的仓库

hexo 的本地安装

npm install hexo-cli -g
hexo init <blogname>
cd blog
npm install
hexo server

使用 ==hexo server== 命令就可以初始一个博客

框架的基本结构与打包

生成的 public 文件夹就直接当成静态网站进行部署即可

hexo 更改本地的主题

首先在

Themes | Hexo 中选择对应的主题，点击头像进入对应主题的 gitHub ，然后进行如下的步骤即可

搭建与发布

  • 博客初始化
  • 本地预览
  • 配置站点
  • 推送站点

写文章使用 markdown ，安装 markdown 编辑器，比如 Typora

文章放在 post 文件夹中，新建文章使用命令

hexo new 文件名

新建专栏使用命令——专栏是显示在侧边的专属栏目

hexo new page 专栏名

同时注意使用插图的时候不能直接连接到本地目录，需要使用图床防止插图，推荐使用：
imgchr.com 和 sm.ms，合规图片会永久保留
